slide 4 · cta

what changed

  • cherry bomb one = logo only. only the blaztly wordmark uses it. everywhere else: space grotesk for display, dm sans for body, caveat for accents.
  • big numbers now space grotesk 700 in orange. no inline font-mixing on the cover hook — the 3 is a separate visual element above the sentence.
  • no tilts. slides, mascot, buttons, type — all straight. the only crooked thing is the scribble arrow (slide 2), which is ink-drawn so it's allowed to be imperfect.
  • cta button is always orange — orange fill, cream text, ink drop-shadow offset for stamp feel.
  • caveat used for: slide tags ("blaztly tips · 01"), footer cues ("swipe →"), slide metadata labels. feels like handwritten notes in the margins.
  • orange highlighter marks one key phrase per slide. never more. the highlight picks the word the reader should remember.
  • paper grain is a subtle svg noise filter on every slide — barely visible, but it's what keeps it from feeling flat/digital.
